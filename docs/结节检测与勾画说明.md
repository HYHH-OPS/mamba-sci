# 传入新 CT 后自动检测并勾画结节

## 一键流程（推荐）

本仓库已实现 **「新 CT → 自动检测（nnU-Net）→ 自动勾画」** 的一键脚本：

```powershell
python scripts/ct_to_detection_and_contour.py --image "D:/path/to/new_ct.nii.gz" [--output_dir "D:/mamba-res/nodules"]
```

- **Step 1**：调用 nnUNetv2 对传入的 CT 做 2D 结节分割预测，得到 mask（并保存为 `output_dir/predicted_mask.nii.gz`）。  
- **Step 2**：用本仓库的勾画逻辑生成轮廓叠加图 `overlay_contour.png` 与结节统计 `nodules.csv`。

**前提**：已安装 [nnUNetv2](https://github.com/MIC-DKFZ/nnUNet)，且 `config/paths.yaml` 中 `nnunet_results`、`nnunet_preprocessed`、`nnunet_raw` 指向正确（或已设置对应环境变量），Dataset503_TBLesion_327 的 2D 模型已训练好。

**若已有 mask、只做勾画**：

```powershell
python scripts/ct_to_detection_and_contour.py --image "CT路径" --skip_nnunet --mask "mask路径" --output_dir "D:/mamba-res/nodules"
```

---

## 当前模型能做什么、不能做什么

| 能力 | 说明 |
|------|------|
| **报告生成（当前已有）** | 传入一张 CT（或取一帧 2D slice）→ **VLM（本仓库训练的模型）** → 生成文字报告（所见、结论、建议、病理倾向）。**不输出结节位置或 mask**。 |
| **结节勾画（需要 mask）** | `scripts/nodule_overlay_and_stats.py` 可根据 **已有分割 mask** 在 CT 上画出结节轮廓并统计。**必须同时传入 --image 和 --mask**，mask 需事先存在。 |
| **自动检测+勾画（已实现）** | `scripts/ct_to_detection_and_contour.py`：新 CT → nnUNetv2 预测 mask → 自动勾画，见上文一键流程。 |

---

## 手动分步（可选）

若不想用一键脚本，可手动执行：

1. **用 nnUNetv2 对新 CT 做分割预测**  
   - 安装 nnUNetv2，设置 `nnUNet_results`、`nnUNet_preprocessed`、`nnUNet_raw`。  
   - 使用 Dataset503 的 2D 配置与 fold_0 权重，得到预测 mask（如 `*_seg.nii.gz`）。

2. **用本仓库脚本勾画**  
   ```powershell
   python scripts/nodule_overlay_and_stats.py ^
     --image "新CT的路径.nii.gz" ^
     --mask "预测得到的mask路径.nii.gz" ^
     --output_dir "D:/mamba-res/nodules/某病例"
   ```

---

## 小结

- **一键流程**：`scripts/ct_to_detection_and_contour.py --image <新CT路径>`，需已安装 nnUNetv2 并配置好 Dataset503 的 2D 权重与路径。  
- **仅勾画**：已有 mask 时用 `--skip_nnunet --mask <mask路径>`，或直接调用 `nodule_overlay_and_stats.py`。

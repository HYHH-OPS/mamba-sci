## 二、在您现有主文档中的改法

### 1. 插入位置（唯一需要动主文档的地方）

在 **“四、针对已有工作的改进与对比实验建议”** 整节之后、**参考文献 `\begin{thebibliography}` 之前**，加入“五、实验设置与结果”：

- **方式 A（推荐）**：主 .tex 与 `paper_section_experiments.tex` 放在同一目录（例如都在 `docs/`）时，在合适位置写：
  ```latex
  \input{paper_section_experiments}
  ```
- **方式 B**：把 `paper_section_experiments.tex` 里从 `\section*{五、实验设置与结果...}` 到文件末尾的全部内容，复制粘贴到主文档同一位置。

### 2. 图片路径

- 若主 .tex 在 **`docs/`** 下，且图片在 **`docs/figures/`** 下，则无需改：实验节里已写为 `figures/train_loss_curve` 等，编译时会到 `figures/` 下找对应 `.png`。
- 若主 .tex 在别处（例如项目根目录），请任选其一：
  - 把 `docs/figures/` 里的 4 张图复制到您主 .tex 所在目录的 `figures/` 子目录，并在主文档 `\begin{document}` 后加：
    ```latex
    \graphicspath{{./figures/}}
    ```
    同时把实验节中的 `figures/xxx` 改为 `xxx`（仅文件名）。
  - 或保持实验节中的 `figures/xxx` 不变，把 4 张图放到主 .tex 同级的 `figures/` 下，编译时在“主 .tex 所在目录”执行 xelatex。

### 3. 编译

在 **主 .tex 所在目录** 执行：

```bash
xelatex paper_with_figures_main.tex
```

或对您自己的主文档：

```bash
xelatex 您的主文档名.tex
```

# SCI 代码框架：保留与删除清单

本仓库已精简为可上传 GitHub 的 SCI 代码框架，删除内容为未使用或环境/一次性脚本与文档。

---

## 保留的核心结构

### 根目录

- **train.py**：Stage 1 训练（Vision + Bridge，proxy loss）
- **train_vlm.py**：Stage 2 训练（图像→报告 caption loss，Mamba 冻结）
- **inference.py**：单张 CT → 报告生成；支持 `--val_sample`、`--draw_nodule_contour`
- **run_full_train.ps1** / **run_full_train_then_infer.ps1**：一键训练与训练+推理脚本
- **ARCHITECTURE.md**：架构说明
- **README.md**：使用说明与示例

### 模块与数据

- **config/paths.yaml**：路径与超参配置
- **data/medical_vlm_dataset.py**：数据集与 `load_paths_config`
- **model/forward_medical_vlm.py**、**model/__init__.py**：VLM 模型
- **vision/nnunet_encoder.py**、**vision/nodule_contour.py**、**vision/__init__.py**：视觉编码与结节轮廓
- **bridge/vim_bridge.py**、**bridge/cmi_connector.py**、**bridge/__init__.py**：桥接与 CMI
- **llm/mamba_loader.py**、**llm/__init__.py**：Mamba 加载

### 脚本（scripts/）

| 脚本 | 用途 |
|------|------|
| **nodule_overlay_and_stats.py** | 图像 + mask → 轮廓叠加图与结节统计 |
| **ct_to_detection_and_contour.py** | 新 CT → nnU-Net 分割 → 勾画（一键） |
| **ct_full_pipeline.py** | 新 CT → 分割+勾画 → 报告 → 侵润倾向占位（全流程） |
| **infer_invasiveness_from_report.py** | 从报告文本做侵润/风险倾向占位推断 |
| **run_ct_to_contour.py** | 仅勾画入口（有 mask 时调用 nodule_overlay_and_stats） |
| **template_caption_csv.py** | 将结构化 CSV 转为模板格式 |
| **excel_total_to_caption_csv.py** | 从 Excel/总表生成 caption CSV |
| **clean_caption_csv.py** | 清洗 caption CSV |
| **validate_dataset.py** | 校验数据集与路径 |
| **inspect_checkpoint.py** | 查看 .pt 中 keys 与 shape |

### 文档（docs/）

- **DATA_PATHS.md**：数据与路径说明
- **TRAIN_COMMANDS.md**：训练命令与参数
- **TRAIN_AND_TEST.md**：训练与测试流程
- **WINDOWS_CONDA_TRAIN.md**：Windows/Conda 训练环境说明
- **训练与幻觉问题分析与解决方案.md**：报告生成问题与对策
- **结节检测与勾画说明.md**：结节分割与勾画流程
- **新CT全流程与侵润等级说明.md**：新 CT 全流程与侵润占位说明
